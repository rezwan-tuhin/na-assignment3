JavaScript Questions

1. What is the difference between let, const and var in JavaScript?

Ans: The differences between let, const and var are in their scope, re-declaration and value assignment. var is function scopped, that means if a var is declared in a function, it is available in all the blocks in that function.

let and const are block scopped, that means, let and const are available only in the block they are declared.

var can be re-declared and reassigned within its scope. let cannot be re-declared but can be reassigned within the same scope. const must be initialized at declaration.

2. What is the difference between find(), filter() method?

Ans: find() and the filter() are two array method in JavaScript. They both search for elements in an array based on certain condition. But there are differences in their behavior. find() search for the first element which satisfies the conditions and returns it, where filter() search for all the elements that meet the condition and returns an array of those elements.

3. How do you select an element in the DOM using JavaScript? Name a few methods.

Ans: In JavaScript, we use several methods of document object to select elements in the DOM. For example,
to access an element by it's id, we use document.getElementById('id');
to acces elements by their class name, we use document.getElementsByClassName('classname');
to access elements by their tag name, we use document.getElementsByTagName('tagname');
